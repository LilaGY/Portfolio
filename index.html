Younited

Web Design

Younited est une organisation fictive de financement collaboratif. L'objectif était de créer les interfaces graphiques d’une application sur smartphone pour récolter des aides financières ou matérielles d’un projet à imaginer.

Pour designer toute l'application il m'a fallu :
• Créer l’identité visuelle de Younited (logos, couleur, typographie..)
• Créer des personas permettant de définir la cible et les fonctionnalités de mon application
• Élaborer et designer les Wireframes

Ombrocessing : Masterclass

Programmation & Sérigraphie

Ombro-cinéma + Processing = Ombrocessing


Ombrocessing est un atelier de création graphique mêlant programmation générative, technique d’ombrocinéma et sérigraphie. Il est mené conjointement par Émeline Vivier de l’Atelier SerreJoint et Julien Gachadoat du studio 2Roqs avec les 10 étudiants du mastère Design Interactif à l’ECV Bordeaux. Un premier atelier a eu lieu du 5 au 13 avril 2018 avec les étudiants de 5e année qui ont été invités à produire une micro-édition de 20 livrets sérigraphiés regroupant des animations abstraites générées par du code avec l’environnement Processing.

Flast

Web Design

Il nous est demandé d'explorer de nouveaux types de transactions auprès d’un distributeur bancaire (ATM) via une application web mobile. S’affranchir de la saisie sur le clavier de l’ATM en effectuant une saisie plus simple sur son téléphone mobile.
L’application web mobile doit permettre à l’utilisateur de localiser un ATM proche, de définir la somme qu’il souhaite retirer ; un code unique est généré pour la transaction. À proximité de de la machine l’utilisateur reçoit un prompt pour s’identifier dans une interface web. Une fois l’identification, faite l’ATM émet les billets...

Ma contribution au sein de l'équipe : Création des wireframes

L’équipe : Lila GUEURY, Morgane MESTRALLET-MARECHAL, Céline MORENO, Charlotte MAGNIN, Victorine RONZON-JARICOT, Clément TERRADE

Abécédaire bestiaire

Typographie

Le brief : choisir une typographie et l'habiller en s'inspirant du monde animal.

Le graphisme de la Phant’ Type est inspiré de l’éléphant.
Nous sommes nombreux à percevoir ce dernier comme un simple être volumineux et pourtant chaque partie de son corps cache bien des secrets que l’on ignore...
Cet alphabet représente point par point les caractéristiques spécifiques de cet animal : sa peau fripée, son regard triste, ses défenses, ses ongles abîmés, sa bouche et bien d’autres choses. Chaque lettre a pour but de faire ressentir chaque texture que l’on peut retrouver chez lui dans les moindres détails : lisse, frippé, abîmé...

TicType :
Workshop
avec Jack Usine

Typographie

Tictype est une typographie réalisée lors d'un workshop avec le typographe Jack Usine. Nous devions créer une police de caractère fondée sur les formes d'un objet choisi (ici, c'est un réveil). À partir de ce dernier, nous avons du analyser chaque partie, chaque détail, afin de relever les formes qui serviraient de base pour la création des caractères.

L'équipe : Lila GUEURY, Marie ZUCCHETTO
