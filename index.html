Ombrocessing : Masterclass

Programmation & Sérigraphie

Ombro-cinéma + Processing = Ombrocessing


Ombrocessing est un atelier de création graphique mêlant programmation générative, technique d’ombrocinéma et sérigraphie. Il est mené conjointement par Émeline Vivier de l’Atelier SerreJoint et Julien Gachadoat du studio 2Roqs avec les 10 étudiants du mastère Design Interactif à l’ECV Bordeaux. Un premier atelier a eu lieu du 5 au 13 avril 2018 avec les étudiants de 5e année qui ont été invités à produire une micro-édition de 20 livrets sérigraphiés regroupant des animations abstraites générées par du code avec l’environnement Processing.

Flast

Web Design

Ce projet consiste à explorer les possibilités d’une nouvelle façon d’opérer les transactions avec un distributeur au moyen d’une application web mobile. L’idée de départ est est la suivante : On élimine le clavier de saisie sur l’ATM et on le remplace par le clavier du téléphone mobile. L’application web mobile permet à l’utilisateur de localiser un ATM proche, de définir à l’avance la somme qu’il souhaite retirer ainsi qu’un code unique pour la transaction. Lorsqu’il s’approche de la machine il reçoit un prompt pour s’identifier dans une interface web. Une fois l’identification faite il prend ses billets .. simple non? peut-être pas tant que ça.

Équipe : MESTRALLET-MARECHAL Morgane, MORENO Céline, MAGNIN Charlotte, RONZON-JARICOT Victorine, TERRADE Clément

Abécédaire bestiaire

Typographie

Sur la base d'une typographie existante, il était demandé d'y apporter un côté "bestiaire"".
Pour cela, grâce à différents éléments qui composent un animal, il fallait réussir à faire ressortir les caractéristiques physique de ce dernier. J’ai, pour ma part, choisi la typographie "Cooper Black". J'ai très vite associé ses empattements à un éléphant. Chaque lettre représente une partie de l'animal : sa trompe, ses pattes, ses yeux, sa peau, etc...

Typographie

Tictype est une typographie réalisée lors d'un workshop avec le typographe Jack Usine. Nous devions créer une police de caractère basée sur les formes d'un objet choisi (ici, c'est un réveil). À partir de ce dernier, nous avons du analyser chaque partie, chaque détail, afin de relever les formes qui nous serviraient pour nos lettres.
